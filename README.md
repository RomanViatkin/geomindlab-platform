# GeoMindLAB Platform

GeoMindLAB is a spatial intelligence platform designed to transform heterogeneous geospatial data into decision-ready analytical outputs.

The platform integrates UAV, LiDAR, satellite, and GIS data into a unified analytical system for territorial diagnostics, infrastructure monitoring, and resilience assessment.
